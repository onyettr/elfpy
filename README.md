# elfpy - ELF file parser

Parses and disaplys ELF (or AXF) file generated from gcc or ARMClang.

## Running
```
$ python3 elfpy.py -h  
$ python3 elfpy.py -f <my ELF file>  
```

## Options
-f      <filename>
-e      Show ELF Header information
-s      Show SECTION information
-p      Show PROGRAM information
-V      Version
-v      Verbose output
